
Directory content
-----------------

This directory contains a Java application sample using RealPro API.

	dsynts.dss ................. DSyntS file sample using DSS file format
	make.bat ................... DOS-Windows script compiling the Java applications
	make.unix .................. Unix script compiling the Java applications
	run.bat .................... DOS-Windows script running the RealPro_DSS_File_App application 
	run.unix ................... Unix script running the RealPro_DSS_File_App application 
	RealPro_DSS_File_App.java .. Java application using RealPro API.
	RealPro_DSS_File_App.class . Compiled RealPro_DSS_File_App application using RealPro API.
	README.txt ................. This file


Instruction for Windows NT/98/2000/XP:
-------------------------------------

To run the Java application under Windows, run the command:

	run.bat 
	
To compile the Java application under Windows, run the command:

	make.bat 


Instruction for Unix or Linux:
-----------------------------

To run the Java application under Unix or Linux, run the command:

	run.unix 
	
To compile the Java application under Unix or Linux, run the command:

	make.unix
