Installation

This section describes how to install RealPro from the ZIP file distribution file RealPro-X.Y.zip, where X.Y is a version number.

In order to install this version of RealPro follow these steps:

  1. Determine the Installation Platform;

  2. Extract the content of the RealPro distribution file;

  3. Set your RealPro serial number;

  4. Set your Java class paths to RealPro Java libraries;

  5. Test your installation.


Determine the Installation Platform

The RealPro distribution file RealPro-X.Y.zip can be installed on platforms running any of the following operating systems:
  • Microsoft Windows 98/NT/2000/XP
  • Linux
  • Unix


Extracting the RealPro Distribution File

The RealPro distribution file can be extracted using standard extraction utilities supporting the ZIP format. Sun's Microsystem Java Development Environment (see RealPro's Software Requirements) provides a jar utility that can be used to extract the content of ZIP file as follow:
jar   -xvf   RealPro-X.Y.zip
Determine an extraction directory where to extract the RealPro distribution file RealPro-X.Y.zip. The full path of this installation directory should not contain any spaces since this can cause some problems with locating files.

Extract the content RealPro distribution file RealPro-X.Y.zip in the extraction directory using a ZIP utility supporting long file names. This will create a directory named RealPro-X.Y.

In the remaining of this document, the term RealPro-Installation-Directory refers to the full path of the extracted directory RealPro-X.Y (e.g. c:\myapps\RealPro-X.Y on MS Windows, /myapps/RealPro-X.Y on Linux);


Setting your RealPro Serial Number

RealPro must be configured with a serial number needed to use RealPro API and RealPro utilities. This serial number is generally sent by e-mail and consists in an alpha-numerical string with the following pattern:

WWWW-XXXX-YYYY-ZZZZ

After receiving your RealPro serial number, configures your RealPro installation in the following way:
  1. Go in the directory RealPro-Installation-Directory/lib (i.e. the subdirectory lib found under the directory RealPro-X.Y where RealPro was extracted).

  2. Edit the file RealPro.properties in the following way: 

    Replace the line: RealPro_SN = WWWW-XXXX-YYYY-ZZZZ

    
By:               RealPro_SN = Your-RealPro-Serial-Number
    where Your-RealPro-Serial-Number is the RealPro serial number that you have received by e-mail.


Setting Java Class Paths to RealPro Java Libraries

In order to use RealPro API or RealPro utilities, the six following items must be in your Java class paths (the three Apache Software Foundation's libraries are standard libraries used for RealPro API):

Path-to-RealPro-Installation-Directory/lib Path to RealPro properties file RealPro.properties where the RealPro serial number is defined.
Path-to-RealPro-Installation-Directory/lib/rpw.jar RealPro engine library
Path-to-RealPro-Installation-Directory/lib/rpw-lkb.jar RealPro knowledge base library
Path-to-RealPro-Installation-Directory/lib/xalan.jar Apache Software Foundation's Xalan library
Path-to-RealPro-Installation-Directory/lib/xercesImpl.jar Apache Software Foundation's Xerces library
Path-to-RealPro-Installation-Directory/lib/xml-apis.jar Apache Software Foundation's XML library

where Path-to-RealPro-Installation-Directory is the absolute or relative path to the directory RealPro-X.Y found in the path RealPro-Installation-Directory.

For example, to run a Java application using RealPRo, the following scripts can be used:

On Windows Platform 

set DIR=c:/myapps/RealPro-X.Y
set CLASSPATH=%DIR/lib;%DIR/lib/rpw.jar;%DIR/lib/rpw-lkb.jar;%DIR/xalan.jar;%DIR/xml-apis.jar;%DIR/xercesImpl.jar
java -classpath  %CLASSPATH  YourJavaRealProApplication

On Unix Platform 

#!/bin/sh
setenv DIR /user/home/myself/myapps/RealPro-X.Y
setenv CLASSPATH $DIR/lib:$DIR/lib/rpw.jar:$DIR/lib/rpw-lkb.jar:$DIR/xalan.jar:$DIR/xml-apis.jar:$DIR/xercesImpl.jar
java -classpath  $CLASSPATH  YourJavaRealProApplication

See the Java documentation for alternative configuration of the class paths.

Note that when running Java graphical applications such as RealPro Workshop, one of RealPro utility, the look and feel may differ from one Java setting to another, on Unix platform in particular.


Testing Your Installation

You can test your installation by running the following script found under the RealPro-Installation-Directory:

rpw.bat (for Microsoft Windows platform)

rpw.unix or sh rpw.unix (for Unix or Linux platform)

If your installation is completed, running the above script will start, RealPro Workshop, a RealPro Java utility for editing Deep-Syntactic Structures, and open its GUI on the screen.
