Platform Configuration

RealPro runs on the following platforms:

  • Operating System: MS Windows 98/NT/2000/XP, Linux or Unix

  • Disk Space: 7.5 Mb

  • Processor speed and memory for PC: 300 Mhz processor (or higher), 128 Mb RAM (or more)

  • Java Development Environment: See below


Java Development Environment

To use RealPro API or RealPro utilities, you must have installed a Java development environment that will let you compile and run Java applications.

Here is a list of recent versions of free Java development environments from Sun Microsystems recommended:

  • Java 2 Platform, Standard Edition (J2SE), 1.4 or higher (J2SE 1.4.2 tested)

    • Note that when running Java graphical applications such as RealPro Workshop, one of RealPro utility, the look and feel may differ from one Java setting to another, on Unix platform in particular. If the look and feel of a Java graphical application does not seem good, modifying the Java setting may give better results.
